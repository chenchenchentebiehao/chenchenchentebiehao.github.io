null集合set（）在python中的方法

  1. set是一个无序不重复的序列
  2. 可以用 { } 或者 set( ) 函数创建集合
  3. 集合存放不可变类型（字符串、数字、元组）

**注意：创建一个空集合必须用 set( ) 而不是 { } ，因为 { } 是用来创建一个空字典**
